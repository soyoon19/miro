# 강화학습 미로 탐험 시뮬레이터

교육용 강화학습 시뮬레이터 웹앱입니다. 학생이 직접 행동을 선택하고 정책을 평가하면서 강화학습의 핵심 개념을 학습할 수 있습니다.

## 📘 프로젝트 개요

이 프로젝트는 강화학습의 핵심 개념:
- **State** (상태)
- **Action** (행동)
- **Reward** (보상)
- **Policy** (정책)
- **Episode** (에피소드)

를 미로 탐험 시뮬레이션을 통해 학습하는 교육용 웹앱입니다.

## 🎮 게임 규칙

### 미로 설정
- **크기**: 4 × 4 그리드
- **시작 위치**: (0, 0)
- **출구 위치**: (1, 3) - 시각화되지 않음
- **장애물 위치**: (3, 0), (1, 1), (0, 3) - 시각화되지 않음

### 행동
- `up`: 위로 이동
- `down`: 아래로 이동
- `left`: 왼쪽으로 이동
- `right`: 오른쪽으로 이동

### 보상 규칙
- 미로 밖 이동 시도 → 제자리, **-5점**
- 장애물 이동 시도 → 제자리, **-5점**
- 정상 이동 → 이동 성공, **-1점**
- 출구 도착 → **+10점**, 즉시 에피소드 종료

### 게임 흐름
- 총 **3개 에피소드**
- 각 에피소드 최대 **12 step**
- 목표 도달 시 즉시 해당 에피소드 종료
- 에피소드 종료 시 미로 위치는 (0,0)으로 초기화
- Q-Table 기록은 유지

## 📊 Q-Table

각 step마다 다음 정보가 기록됩니다:
- 에피소드/Step
- State (현재 위치)
- Action (선택한 행동)
- 이동 가능 여부
- Reward (보상)
- Next State (다음 위치)
- Total Score (누적 점수)
- Policy (정책 문자열)
- 탐험가 의견 반영 여부 (O/X) - **학생 입력 필수**
- 정책 판단 (좋은 정책/나쁜 정책) - **학생 입력 필수**

## 🧠 학습 목표

학생은 매 step마다:
1. 행동을 선택하고
2. 이동 가능 여부 피드백을 받고
3. 보상을 확인하고
4. 탐험가 의견 반영 여부를 선택하고
5. 정책을 평가하고
6. 모든 기록을 Q-Table에서 확인합니다

게임 종료 시:
- **탐험가 의견 반영률**이 계산되어 표시됩니다.

## ⚠️ 중요 사항

이 프로젝트는:
- ❌ 강화학습 자동 구현이 아닙니다
- ✅ 인간이 정책을 평가하는 **Human-in-the-loop RL 교육 시뮬레이터**입니다

학생은 출구와 장애물의 위치를 모른 채 탐험하며, 매 행동마다 직접 판단하고 기록하면서 학습합니다.
